# Snake Game

A realistic snake game built using HTML, CSS, and JavaScript.

## Features

- Smooth snake animation
- Score tracking
- High score storage
- Obstacles
- Particle effects
- Sound effects
- Responsive design

## Controls

- Arrow Keys / WASD - Move
- P - Pause
